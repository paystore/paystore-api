# paystore-api
Biblioteca para integração com aplicativo de pagamentos da PayStore.

A documentação da API encontra-se [aqui](http://177.69.97.18:6655).



#### Compatibilidade entre payments-api e Paystore Payments App
| Versão aar              | Versões do Payments app que são compatíveis (v2.11.x)| Versões do Payments app que são compatíveis (v2.12.x) |
| ----------------------- | -------------------------------             | -------------------------------             |
| `payments-api-2.10.6.8`   | v2.10.6.13                                  | N/A                                         | 
| `payments-api-4.0.1.4`    | N/A                                         | v2.12.3.10 a v2.12.3.12                     | 
| `payments-api-4.0.2.8`    | v2.11.7.7 a v2.11.7.14                      | v2.12.3.13 a v2.12.3.18                     | 
| `payments-api-4.1.0.1`    | v2.11.7.15                                  | v2.12.3.19 a v2.12.3.20                     | 
| `payments-api-4.1.0.3`    | v2.11.8 a v2.11.9                           | v2.12.7                                     |
